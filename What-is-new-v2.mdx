description: "New features and changes introduced in Kontratar version 2.0."
---

Version 2.0 introduces five new modules that extend Kontratar from opportunity tracking and proposal generation into a full-lifecycle government business development platform.

## New modules

| Module | What it does |
| --- | --- |
| [**Awards**](/Version-2/Awards) | Browse, filter, and analyze federal contract awards sourced from USAspending.gov. Includes award analytics, entity profiles, AI Chat, Projects, and scheduled Research. |
| [**Forecasts**](/Version-2/Forecasts) | Browse planned future procurements that agencies have announced but not yet solicited. Includes calendar view, AI Chat, Projects, and scheduled Research. |
| [**eBuy Opportunities**](/Version-2/eBuy-Opportunities) | View GSA Schedule opportunities from the eBuy portal alongside SAM.gov opportunities. Includes AI-extracted requirements, partner ranking, and proposal outlines. |
| [**Campaigns**](/Version-2/Campaigns) | Create email templates and send bulk outreach to contracting officers and teaming partners. Includes attachment token personalization and a 21-day government email cooldown rule. |
| [**Pipeline Management**](/Version-2/Pipeline-Management) | Track opportunities through an 8-stage capture management lifecycle with dashboard metrics, funnel visualization, and stage-by-stage management. |

## New AI capabilities in v2.0

- **AI Chat for Awards and Forecasts** — Query the awards and forecast databases in plain English. Ask questions like "What is the highest-valued award in the database?" and receive data-backed answers.
- **Projects** — Create scoped research workspaces with custom instructions and file uploads. The AI answers only within the context you define.
- **Scheduled Research** — Set up recurring AI queries (daily, weekly, monthly, yearly) that run automatically. Results are saved by date for trend comparison and can be exported as Word, Excel, or CSV.

## Changes from v1.2

All features from v1.2 remain available. Version 2.0 adds the modules listed above without removing or changing existing functionality. The sidebar has been reorganized to accommodate the new sections.
